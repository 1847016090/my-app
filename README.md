## Vue + Vant + Mongodb + koa
### Init our project
1. **Install vue-cli**
`npm install -g vue-cli`
2. **use cli to create a project**
`vue init webpack front`
if your vue-cli version is more than 3.0.0, you can:
`vue create front`
also you can:
    ```
    npm install add @vue/cli-init -g

    vue init webpack front
    ```
